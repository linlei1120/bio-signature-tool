======================================
手写签名提取工具 - 使用说明
======================================

【快速开始】
1. 双击运行 "启动程序.bat"
   或
   在命令行运行: python signature_extractor.py

2. 选择功能模式:
   - 模式1: 从图片文件提取签名
   - 模式2: 实时摄像头拍照提取签名
   - 模式3: 批量处理文件夹中的图片

【三种使用模式详解】

模式1 - 从图片文件提取签名
-------------------------
1. 输入图片文件路径（支持JPG、PNG、BMP等格式）
2. 选择是否使用自定义阈值（建议先用默认的自动阈值）
3. 程序会自动提取签名并保存到 extracted_signatures 文件夹
4. 显示预览窗口，按任意键关闭

模式2 - 实时摄像头拍照提取签名
-----------------------------
1. 程序会打开摄像头实时预览
2. 将签名纸放在摄像头前
3. 按键操作：
   - 按 'c' 或 '空格' → 拍照并提取签名
   - 按 's' → 保存当前提取的签名
   - 按 '+' → 增加阈值（使签名更细）
   - 按 '-' → 减少阈值（使签名更粗）
   - 按 'q' 或 'ESC' → 退出

模式3 - 批量处理文件夹
---------------------
1. 输入包含图片的文件夹路径
2. 选择是否使用自定义阈值
3. 程序会自动处理文件夹中所有图片
4. 提取的签名都保存到 extracted_signatures 文件夹

【输出文件】
所有提取的签名都保存在 extracted_signatures 文件夹中
- 从文件提取: 原文件名_signature.png
- 实时拍照: signature_年月日_时分秒.png
- 格式: 透明背景的PNG图片

【使用技巧】
1. 最佳效果: 深色签名 + 浅色背景 + 良好光照
2. 如果签名太粗: 增加阈值（按 + 键）
3. 如果签名太细: 减少阈值（按 - 键）
4. 背景越简洁，提取效果越好

【常见问题】
Q: 摄像头无法打开？
A: 检查摄像头是否被其他程序占用，关闭后重试

Q: 未检测到签名？
A: 尝试调整阈值，或确保签名和背景有足够对比度

Q: 提取效果不理想？
A: 使用高质量图片，保持良好光照，背景尽量简洁
